# LYSIN

Jump and Run

Ziel: an das Ende kommen / kein Ziel / Checkpoints (statistik / Highscore) / Zeitlimit (Timer) / (Level (Sterne / Bewertungen))

Figur: Laufen / Sprinen / Leben / Level, Erfahrung / verschiedene (individuelle) Charaktere / HP /(geschwinigkeit/ sprunghöhe/weite                                                         nach größe)

Items: schneller Laufen / höher Springen / mehr (extra) Leben / HP auffüllen

Hindernisse: Löcher / Fallen / Bouncepad / bewegende Plattformen / Rätsel 

Neg. Items: blindheit / langsamer / nicht so hoch springen 

Welt: Story / Aussehen

Währung ---> Upgrades von Charaktern / Einstellungen (Tastatur / Joystick)
